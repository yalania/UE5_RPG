UE5_RPG
A work-in-progress action RPG built in Unreal Engine 5, heavily inspired by Bloodborne. This project is a personal learning exercise focused on three main areas: Unreal Engine 5 development, the Gameplay Ability System (GAS), and AI behavior.
As a learning project, expect things to be rough around the edges. The goal is growth, not polish.
Learning Goals

Unreal Engine 5 core systems and workflows
Gameplay Ability System (GAS) architecture and implementation
AI design and behavior trees

Status
Work in progress. Ongoing development.
